Detailed Description

Represents a PARAMETER block in the NMODL.

Variables whose values are normally specified by the user are parameters and are declared in a PARAMETER block. Here is an example :

1 PARAMETER {
2  gkbar=.01 (mho/cm2) : Maximum Permeability
3  d1 = .84
4  k2 = .13e-6 (mM)
5  abar = .28 (/ms)
6  lcai (mV)
7 }

All parameters are stored in the ast::ParamBlock::statements as vector.
